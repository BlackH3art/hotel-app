# Hotel-app

![hotel-app]()

Simple React-app built with: 
- typescript
- tailwind.css
- axios
- framer-motion

Project is initialized with:
- Vite

---

### How is this working?
Application is fetching hotels data and allow users to filter it by star rating and specified occpancy.
